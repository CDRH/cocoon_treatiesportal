Search Help

Basic search
Simply enter the word you wish to find and the search engine will seek out every instance of the word in these nine documents. For example: treaty. All instances of the use of the word treaty will show up on the search results page. The search does not support wildcards (*) at this time.

Excluding a word
You can exclude a word from a search by using a minus (-) symbol in front of it. For example, "treaty -Iroquois" will search for instances of the word "treaty," but will not include documents with the word "Iroquois."

Capitalization
Searches are not case sensitive. For example: six nation will produce the same results as Six Nation.
